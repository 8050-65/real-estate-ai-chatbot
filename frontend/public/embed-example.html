🤖 Leadrat AI Chatbot Widget

Production-ready embed script for real estate websites

✅ Latest Version - All features working in production

⚡ Quick Start

Add this to your HTML (before closing </body> tag):

<script async src="https://real-estate-ai-chatbot.pages.dev/chatbot-embed.js?v=20260504" data-chatbot-url="https://real-estate-ai-chatbot.pages.dev" data-api-url="https://real-estate-api-dev.onrender.com/api/v1/chat/message"> </script>

📋 Configuration Options

Customize via data attributes:

  • data-chatbot-url: Frontend widget URL (where chat UI is hosted)
  • data-api-url: Backend API endpoint (for messages)
  • data-position: bottom-right | bottom-left | top-right | top-left (default: bottom-right)
  • data-theme: dark | light (default: dark)

🧪 Browser Console Test

After loading the page, check the console:

window.LeadratChatConfig

Should show:

{ chatbotUrl: "https://real-estate-ai-chatbot.pages.dev", apiUrl: "https://real-estate-api-dev.onrender.com/api/v1/chat/message", position: "bottom-right", theme: "dark" }

🔧 Programmatic Control

window.leadratChatbot.open() window.leadratChatbot.close() window.leadratChatbot.toggle()
