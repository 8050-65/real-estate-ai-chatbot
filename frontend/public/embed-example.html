Quick Start (3 Steps)

Step 1: Copy the script tag below

<script async src="https://real-estate-ai-chatbot.pages.dev/chatbot-embed.js" data-chatbot-url="https://real-estate-ai-chatbot.pages.dev" data-api-url="https://real-estate-api-dev.onrender.com/api/v1/chat/message" data-position="bottom-right" data-theme="dark"> </script>

Step 2: Paste it into your HTML (before closing </body>)

Step 3: The chatbot appears automatically!

⚙️ Configuration Options

Customize the chatbot behavior with these data attributes:

data-chatbot-url
https://real-estate-api-dev.onrender.com/api/v1/chat/message
Backend URL where chatbot runs
data-position
bottom-right | bottom-left | top-right | top-left
Where button appears (default: bottom-right)
data-theme
dark | light
UI theme (default: dark)

🚀 Testing Locally

To test the chatbot before deploying to production:

<!-- Local Development URL --> <script async src="http://localhost:3000/chatbot-embed.js" data-chatbot-url="http://localhost:3000" data-position="bottom-right" data-theme="dark"> </script>

Requirements:

  • Frontend running on http://localhost:3000
  • Ollama running (http://localhost:11434)
  • Backend running (http://localhost:8001)

✨ What Happens When Users Click

  1. Button Click: Floating button opens chat window with animation
  2. Status Check: Shows Ollama, RAG, and Leadrat connection status
  3. Chat Interaction: User types question or command
  4. Smart Routing:
    • General questions: Uses RAG + Ollama for AI answer (2-5 sec)
    • Structured queries: Routes to Leadrat API for live data (<1 sec)
  5. Response Display: Natural language answer with quick reply buttons
  6. Multi-turn: Remembers conversation history for context

🔧 Troubleshooting

Chatbot doesn't appear?

  • Check browser console for errors (F12 → Console)
  • Verify script URL is correct and accessible
  • Ensure backend chatbot URL is running
  • Check for CORS issues if domains don't match

Status shows offline?

  • Start Ollama: ollama run llama3.2
  • Verify FastAPI backend is running on port 8001
  • Check ChromaDB is seeded with: python scripts/seed_rag.py
