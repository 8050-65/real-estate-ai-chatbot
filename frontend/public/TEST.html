Installation Code

To add the chatbot to your website, add this single line before </body>:

<script async src="http://localhost:3000/EMBED.js"></script>

For Production

When deployed, update the URL to your production domain:

<script async src="https://your-domain.com/EMBED.js"></script>

JavaScript API

Control the chatbot programmatically:

window.leadratChatbot.open() // Open chat window.leadratChatbot.close() // Close chat window.leadratChatbot.toggle() // Toggle open/close

Troubleshooting

Button not showing?

  • Check dev server is running: npm run dev
  • Press F12 → Console tab, look for initialization message
  • Verify http://localhost:3000 loads

Widget not opening?

  • Check F12 console for errors
  • Try clicking the floating button
  • Use window.leadratChatbot.open() in console

Text not readable?

  • Widget should have white background
  • Text should be dark gray/black
  • Try clearing browser cache
