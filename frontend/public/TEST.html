Installation

To add the chatbot to your website, add this single line of code before closing </body> tag:

<script async src="http://localhost:3000/EMBED.js"></script>

For Production

When deploying to production, update the URL:

<script async src="https://your-domain.com/EMBED.js"></script>

Configuration

You can specify a custom chatbot URL (for multi-domain setups):

<script async src="http://localhost:3000/EMBED.js" data-chatbot-url="http://localhost:3000"> </script>

Features

  • Floating button in bottom-right corner
  • Click to open/close widget
  • Auto-closes when clicking outside
  • Works on mobile and desktop
  • Responsive design
  • Light theme with readable text
  • JavaScript API for programmatic control

JavaScript API

Control the chatbot programmatically:

// Open the chat widget window.leadratChatbot.open() // Close the chat widget window.leadratChatbot.close() // Toggle open/close window.leadratChatbot.toggle()

Example: Custom Button

<button onclick="window.leadratChatbot.open()"> Get Help </button>

🔧 Troubleshooting

Button not showing?

  • Check dev server is running: npm run dev
  • Open F12 console, look for initialization message
  • Verify http://localhost:3000 loads in browser
  • Check browser is not blocking popups/scripts

Widget looks broken?

  • Check http://localhost:3000/embed loads properly
  • Check browser console (F12) for errors
  • Try hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
  • Check Network tab for failed requests

Text not readable?

  • Widget should have white background
  • Text should be dark gray/black
  • Input field should have visible placeholder
  • Buttons should have good contrast
  • If not, try clearing browser cache
⚠️ Note: If you see console errors mentioning "SessionProvider", make sure /embed route is using the new EmbedChatWidget component, not ChatInterface.
