📋 Integration Instructions

Once the widget is deployed to your CDN, add this to any website:

<script>
  window.LeadratChatConfig = {
    apiUrl: "https://your-api.com/api/v1/chat",
    botName: "AI Assistant",
    botSubtitle: "Real Estate AI",
    primaryColor: "#6C63FF",
    position: "bottom-right"
  };
</script>
<script src="https://cdn.yourdomain.com/chatbot/leadrat-chat.js" async></script>

The script will automatically inject a floating chat button in the bottom-right corner. No additional setup required!
