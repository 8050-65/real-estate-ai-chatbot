## 🚨 IF YOU GET STUCK

### Service Takes Long Time
→ Normal for free tier (10-15 min first build)
→ Check logs in service dashboard
→ Be patient, don't restart

### 503 Service Unavailable
→ Free tier Render spins down after inactivity
→ Wait 1-2 minutes and refresh
→ Make request to wake up service

### Database Connection Error
→ Check credentials match exactly
→ Verify environment variables
→ Check Render dashboard for errors
→ See RAPID_DEPLOYMENT_VERIFICATION.md troubleshooting

### CORS or API Errors
→ Verify CORS_ALLOWED_ORIGINS set correctly
→ Check API URLs in frontend env vars
→ Check API is actually responding
→ Restart service if needed

### GitHub Secrets Not Working
→ Ensure secret names match exactly
→ Verify secret values are correct
→ Check GitHub Actions can access them
→ Re-add secret if needed
