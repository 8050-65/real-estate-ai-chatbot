📦 Installation Code

Copy this line to embed the chatbot in your website:

Option 1: Basic (Production URL)

<script async src="https://chatbot-leadrat.vercel.app/chatbot-embed.js"></script>

Option 2: With Configuration

<script async src="https://chatbot-leadrat.vercel.app/chatbot-embed.js" data-position="bottom-right" data-theme="dark"> </script>

Option 3: Custom URL

<script async src="https://chatbot-leadrat.vercel.app/chatbot-embed.js" data-chatbot-url="https://your-custom-url.com"> </script>
💡 Place this in your HTML:
Add the script tag just before the closing </body> tag or anywhere after the opening <body> tag.

⚙️ Configuration Options

Attribute Values Default Description
data-chatbot-url URL string chatbot-leadrat.vercel.app Custom chatbot URL
data-position bottom-right, bottom-left, top-right, top-left bottom-right Button position on page

🔧 Troubleshooting

Chat button not showing?

  1. Check browser console (F12 → Console tab)
  2. Verify script URL is correct
  3. Clear browser cache and refresh
  4. Check z-index conflicts with other elements

Chat window blank?

  1. Check if https://chatbot-leadrat.vercel.app/ai-assistant is accessible
  2. Verify network connectivity
  3. Check Network tab in developer console for errors
  4. Ensure CORS is properly configured

Messages not responding?

  1. Check backend API availability
  2. Verify Leadrat API connectivity
  3. Check browser Network tab for failed requests
  4. View console for error messages
