📦 Installation Code (Localhost)

For testing, use this code:

<script async src="http://localhost:3000/chatbot-embed.js"></script>

For Production (when deployed):

<!-- Change to your production URL -->
<script async src="https://your-production-url.com/chatbot-embed.js"></script>

🔍 What's Happening

  1. This page loads the embed script from http://localhost:3000/chatbot-embed.js
  2. The script creates a floating button in bottom-right corner
  3. When you click the button, it opens a chat window
  4. The chat window loads the chatbot app via iframe: http://localhost:3000/ai-assistant?embedded=true
  5. The ?embedded=true parameter hides the REIA header and status bar
  6. You can send messages and the backend responds

🔧 Troubleshooting

  • Button not appearing? → Open F12 console, check for red errors
  • Chat window blank? → Verify http://localhost:3000 is running
  • No response to messages? → Check Network tab, look for failed API calls
  • Console errors? → Check the error message carefully
