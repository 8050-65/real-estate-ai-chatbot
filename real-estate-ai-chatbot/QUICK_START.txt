KEY FIXES DEPLOYED
================================================================================

1. Leadrat API Base URL
   BEFORE: https://connect.leadrat.info/api/v1 (wrong - QA endpoint)
   AFTER:  https://connect.leadrat.com/api/v1 (correct - production endpoint)

2. Tenant-Aware Routing
   - Production tenants (dubait11) → .com endpoint
   - QA/test tenants → .info endpoint
   - Automatic routing in LeadratClient.getBaseUrl()

3. Token Cache Per Tenant
   - BEFORE: All tenants shared "leadrat:token" key → conflicts
   - AFTER: Each tenant gets "leadrat:{tenant}:token" → isolated

4. Frontend Error Handling
   - BEFORE: Shows [API Error] {} to users
   - AFTER: Friendly message + 2 retries + 10s timeout
   - Logs: Detailed debug info for troubleshooting
