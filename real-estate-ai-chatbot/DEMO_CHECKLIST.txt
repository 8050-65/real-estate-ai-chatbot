================================================================================
IF SOMETHING GOES WRONG
================================================================================

CHATBOT DOESN'T RESPOND:
  □ Check health check URLs above
  □ Browser console (F12) - look for errors
  □ Refresh page (Ctrl+R)
  □ Check: Is Ollama online? (green indicator)

CORS ERROR IN CONSOLE:
  □ Likely environment variables not set in Render
  □ Check Render dashboard for ENVIRONMENT=production
  □ Reset browser cache (Ctrl+Shift+Delete)

PAGE WON'T LOAD:
  □ Check Vercel status: https://vercel.com/dashboard
  □ Try incognito mode (Ctrl+Shift+N)
  □ Check internet connection

OLLAMA OFFLINE:
  □ This is expected if backend isn't running
  □ AI questions will fail, but API queries should work
  □ Status bar shows: ✗ Ollama Offline (red)
